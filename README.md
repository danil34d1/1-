# Мыло — каталог сырья и автохимии

Репозиторий хранит проверяемые данные о мыле, моющих средствах, ПАВ, компонентах автохимии, фасовке и источниках сведений.

Главное правило: **не превращать догадку в факт**. Если наименование из сообщения, накладной или этикетки неоднозначно, запись получает статус `needs-verification`, исходный текст сохраняется без исправления, а возможные расшифровки оформляются как гипотезы.

## Структура

- `data/products/` — одна JSON-карточка на продукт или сырьё;
- `schemas/product.schema.json` — формальная схема карточки;
- `assets/product-template.json` — шаблон новой карточки;
- `scripts/` — добавление, проверка каталога и запись подтверждённых уроков;
- `tests/` — автоматические проверки;
- `experiments/` — планы и записи контролируемых опытов;
- `research/` — научные статьи, патентные наводки, оборудование, открытые проекты и очередь проверок;
- `.agents/skills/soap-data-catalog/` — локальный Codex Skill;
- `.agents/skills/used-oil-soap-pilot/` — лабораторный шлюз и расчётная запись для отработанного масла;
- `AGENTS.md` — обязательная маршрутизация задач к skill.

## Исследовательская работа

Начальная карта международных источников находится в `research/first-wave-2026-07-16.md`.

Статусы внешних находок:

- `lead` — наводка;
- `reviewed` — источник изучен;
- `experiment-required` — требуется собственный опыт;
- `confirmed` — вывод подтверждён;
- `rejected` — источник или направление отклонены.

Форум, видео или патент не становятся рабочей рецептурой без проверки.

## Быстрая проверка

```bash
python scripts/validate_catalog.py
python scripts/check_skill.py
python -m unittest discover -s tests -v
```

## Добавление записи

```bash
python scripts/add_product.py \
  --id aes-70 \
  --name "AES 70" \
  --source-type label \
  --source-text "AES 70, 170 кг"
```

Новая запись создаётся как `needs-verification`. Переводить её в `verified` можно только после фиксации достаточного источника: фото этикетки, паспорта качества, SDS/TDS, накладной с однозначным артикулом либо другого проверяемого документа.
