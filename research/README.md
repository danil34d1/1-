# Исследовательский центр проекта «Мыло»

Этот раздел хранит внешний опыт, который может быть полезен для переработки отработанного подсолнечного фритюрного масла (UCO-SF), подготовки сырья и разработки технических моющих продуктов.

## Главное правило

Интернет-находка не становится рабочей технологией автоматически.

Каждой записи присваивается статус:

- `lead` — наводка, которую ещё не проверяли по первичному источнику;
- `reviewed` — источник прочитан, условия и ограничения выписаны;
- `experiment-required` — идея правдоподобна, но требует собственного опыта;
- `confirmed` — вывод подтверждён стандартом, анализом, контролируемым опытом или воспроизводимым тестом;
- `rejected` — источник ошибочен, нерелевантен или технология не подходит проекту.

## Уровни доверия

1. Стандарт, нормативный документ или официальный метод анализа.
2. Рецензируемая научная статья с описанной методикой.
3. Патент — источник конструкции и вариантов процесса, но не доказательство эффективности и не разрешение на использование.
4. Документация производителя оборудования — достоверна для заявленной машины, но требует испытания на нашем сырье.
5. Форум, видео, сообщение или коммерческая статья — только источник гипотез и типичных проблем.

## Структура

- `first-wave-2026-07-16.md` — первый международный обзор источников и практических направлений;
- `action-plan.md` — последовательность работ первого исследовательского цикла;
- `equipment-screening-v1.md` — требования к лабораторному стенду, пилотной ёмкости, фильтрации и будущему запросу центрифуги;
- `laboratory-request-template.md` — единый запрос в лаборатории по RAW/F1/F2/F3;
- `technical-cleaner-test-plan-v1.md` — измеримое сравнение технического мыла с покупным обезжиривателем;
- `patent-search-queries.md` — многоязычные поисковые запросы и правила разбора патентов;
- `../experiments/uco-heating-24h.md` — ранняя карта наблюдений при 24-часовом нагреве;
- `../experiments/uco-separation-screening-v1.md` — контролируемый протокол A/B/C для холодного отстоя, прогрева-отстоя и постоянного нагрева;
- последующие обзоры создаются отдельными датированными файлами;
- подтверждённые общие уроки переносятся в skill только после проверки;
- результаты конкретной партии хранятся отдельно от общих знаний.

## Ближайшие цели исследования

1. Получить первую реальную партию и сохранить контрольную пробу RAW.
2. Выполнить протокол трёх режимов разделения и закрыть материальный баланс.
3. Получить предложения лабораторий и выбрать применимые методы.
4. Определить недорогую схему подготовки: отстой, раздельный слив и ступенчатая фильтрация.
5. Рассчитать мыло только после лабораторного SAP конкретной F1.
6. Испытать первый технический продукт по удалению загрязнения и совместимости с материалом.
7. Собрать патентную карту по подготовке UCO, омылению и трёхфазному разделению.
8. Отдельно вести дальние направления: MES/SME, полимеры, асфальтовые добавки и другие продукты высокой переработки.

## Открытые рабочие задачи

- GitHub Issue #1 — патентный ландшафт минимум из 20 семейств;
- GitHub Issue #2 — сборка безопасного трёхсосудного стенда.

## Что запрещено считать доказательством

- прозрачность масла без анализа воды и примесей;
- хороший запах или цвет;
- усреднённый SAP свежего подсолнечного масла для неизвестной UCO-партии;
- один успешный ролик или форумное сообщение;
- пример из патента без воспроизведения;
- пена как единственный показатель моющей способности;
- успешное омыление как подтверждение безопасности продукта.
