# Лестница продуктов из UCO-SF v1

Дата: 2026-07-16

Статус: `decision framework / economics not yet confirmed`

## 1. Назначение

Этот документ помогает не перескакивать от сырого масла сразу к сложной химии. Каждое направление оценивается по доступности, риску, требованиям к оборудованию, лабораторному контролю и соответствию текущим возможностям проекта.

## 3. Критерии перехода между уровнями

### Уровень 0 → 1

Нужно подтвердить:

- прослеживаемый прямой сбор из кафе;
- отсутствие смешения с жироуловителем и мойкой;
- измеримый выход F1/F2/F3;
- воспроизводимую процедуру отстоя и фильтрации;
- закрытый материальный баланс.

### Уровень 1 → 2

Нужно подтвердить:

- лабораторное число омыления конкретной партии;
- воду и нерастворимые примеси;
- кислотное число;
- чистоту щёлочи;
- малую безопасно контролируемую партию;
- отсутствие непредусмотренных загрязнений.

### Уровень 2 → 3

Нужно подтвердить:

- повторяемость минимум нескольких партий;
- процент удаления стандартного загрязнения;
- сравнение с покупным продуктом;
- отсутствие недопустимой коррозии;
- стабильность при хранении;
- расход и себестоимость рабочего раствора.

### Уровень 3 → 4/5

Нужно подтвердить:

- стабильный поток сырья;
- коммерческий спрос;
- экономику базового продукта;
- наличие лабораторного или промышленного партнёра;
- понятный рынок продукта высокой переработки;
- оценку патентных и регуляторных рисков.

## 4. Рекомендуемый коммерческий маршрут

```text
Шаг 1: научиться принимать и классифицировать UCO-SF
Шаг 2: получать стабильную подготовленную F1
Шаг 3: проверить техническое мыло внутри своей автомойки
Шаг 4: создать составной обезжириватель под одну конкретную задачу
Шаг 5: подтвердить себестоимость и повторяемость
Шаг 6: только затем выбирать биотехнологическое или химическое расширение
```

## 5. Главный экономический показатель подготовки масла

Для каждого режима считать:

```text
Себестоимость подготовки 1 кг F1 =
(энергия + фильтры + труд + анализы + потери + амортизация) / масса годной F1
```

Высокий процент выхода масла сам по себе не означает выгоду, если режим требует большого расхода энергии, быстро забивает фильтры или ухудшает качество партии.

## 6. Текущее решение

Финансирование и закупки направлять только на уровни 0–3. Уровни 4–6 пока финансируются только как поиск литературы, патентов и партнёров.
