# Repository instructions

## Mandatory skill routing

- Если задача связана с добавлением, исправлением, импортом, классификацией, объединением или проверкой данных о мыле, моющих средствах, ПАВ, сырье или автохимии, Codex обязан вызвать `$soap-data-catalog` **до внесения изменений**.
- Правило действует и тогда, когда пользователь описывает ту же конечную цель другими словами, например: «сохрани компонент», «разбери этикетку», «обнови карточку сырья» или «занеси товар из накладной».
- Для задач вне каталога и для общих объяснений без изменения данных `$soap-data-catalog` не применять.
- Работу нельзя считать завершённой, пока не выполнены обязательные проверки из skill.
