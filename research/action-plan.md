# План действий исследования

Обновлено: 2026-07-16

## Этап 1 — определить сырьё

- [ ] Присвоить первой реальной партии код `UCO-SF-...`.
- [ ] Записать кафе, дату, массу, тип исходного масла и продукты, которые жарили.
- [ ] Отобрать и сохранить контрольную пробу до подготовки.
- [ ] Зафиксировать фото, цвет, запах, воду, осадок и температуру.
- [ ] Не принимать в основной поток масло из жироуловителя или после мойки.

## Этап 2 — проверить разделение

- [ ] Подготовить три одинаковых образца одной партии.
- [ ] Провести холодный отстой.
- [ ] Провести прогрев с последующим выключением и отстоем.
- [ ] Провести выдержку при постоянной температуре без механического перемешивания.
- [ ] Измерить объёмы F1, F2 и F3.
- [ ] Отобрать отдельные пробы каждой фазы.
- [ ] Сравнить расход энергии и полезный выход F1.

## Этап 3 — лаборатория

- [ ] Найти лабораторию, выполняющую ISO 660, ISO 662, ISO 663 и ISO 3657.
- [ ] Уточнить требования к объёму пробы и таре.
- [ ] Заказать анализ исходной партии и подготовленной F1.
- [ ] Добавить перекисное число для оценки вреда длительного нагрева.
- [ ] Уточнить методы обнаружения минерального масла и моющих средств.
- [ ] Сохранить номер протокола и дату анализа.

## Этап 4 — конструкция пилотной подготовки

- [ ] Нарисовать ёмкость с коническим или наклонным дном.
- [ ] Предусмотреть нижний слив F3.
- [ ] Предусмотреть отдельный отбор F2.
- [ ] Предусмотреть отбор F1 без захвата промежуточного слоя.
- [ ] Добавить минимум три пробоотборные точки по высоте.
- [ ] Выбрать безопасный косвенный нагрев.
- [ ] Установить рабочий датчик температуры и независимую защиту от перегрева.
- [ ] Выбрать грубое сито и мешочные фильтры для ступенчатого теста.

## Этап 5 — первый технический продукт

- [ ] Не рассчитывать щёлочь до получения лабораторного SAP партии.
- [ ] Выбрать назначение: пол, стальные детали или жирное оборудование.
- [ ] Начать с малой лабораторной партии.
- [ ] Проверить свободную щёлочь и стабильность.
- [ ] Сравнить с покупным обезжиривателем на одинаковых металлических купонах.
- [ ] Измерять удаление загрязнения по массе, а не только пену.
- [ ] Проверить коррозионность для стали и отдельно для алюминия.

## Этап 6 — патенты и внешние технологии

- [ ] Найти первые 20 патентных семей по запросам из `patent-search-queries.md`.
- [ ] Отсеять патенты на возврат масла в пищевое использование.
- [ ] Отдельно выделить конструкции отстойников, сепараторов и реакторов.
- [ ] Проверить правовой статус и семейство пяти наиболее полезных документов.
- [ ] Найти первичную статью 2025 года о полиэфирах/клеях из UCO.
- [ ] Найти современные публикации по UCO-SF, а не только по пальмовому маслу.

## Критерий завершения первого цикла

Первый цикл завершён только когда есть:

1. реальная партия с паспортом;
2. сравнение минимум трёх режимов подготовки;
3. лабораторные показатели до и после;
4. материальный баланс F1/F2/F3;
5. решение, какой технический продукт испытывать;
6. доказуемый тест его эффективности.
