# Проверка автоматического выбора skill

## POSITIVE-1: ACTIVATE
**Запрос:** «Добавь в каталог новое сырьё AES 70 из фото этикетки и сохрани массу бочки».
**Почему:** создаётся карточка ПАВ и переносятся данные из первичного источника.

## POSITIVE-2: ACTIVATE
**Запрос:** «В накладной название мыла написано иначе — найди старую запись, исправь её и проверь, нет ли дубля».
**Почему:** обновляются и объединяются данные каталога, даже без прямого упоминания skill.

## NEGATIVE-1: DO-NOT-ACTIVATE
**Запрос:** «Объясни простыми словами, почему ПАВ образуют пену, но ничего не сохраняй в проект».
**Почему:** это общее объяснение без изменения данных репозитория.
